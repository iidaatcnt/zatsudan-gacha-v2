
# 雑談ネタガチャ v3 (Zatsudan Gacha)

雑談の話題（ネタ）をランダムに提供するWebアプリケーションです。
Googleスプレッドシートで管理された話題データを読み込み、ガチャ形式で表示します。

## ✨ 特徴
- **クリーンなデザイン**: 白を基調とした見やすいUI
- **カスタマイズ機能**: アプリのタイトルやデータソース（スプレッドシート）を自由に設定可能
- **カテゴリ分類**: 「貯める」「稼ぐ」などのカテゴリごとに話題を絞り込み
- **単一選択モード**: 気に入った話題を1つだけ選んでハイライト（ハートマーク）

## 🚀 使い方

### 1. アプリを開く
ブラウザでアプリにアクセスします。
初期状態ではデモ用のデータが表示されます。

### 2. ガチャを回す
「🎲 ガチャ（2ネタ）」ボタンを押すと、ランダムに2つの話題が表示されます。

### 3. 話題を選ぶ
表示された話題をクリックすると、背景色が濃くなり「❤️」マークがつきます。
今日の雑談テーマとして決定した話題を視覚的に固定できます。

### 4. カテゴリで絞り込む
上部のカテゴリボタン（「貯める」「稼ぐ」など）を押すと、そのジャンルの話題だけが出現するようになります。
「ランダム」を選ぶと全ジャンルから抽選されます。

## ⚙️ 設定（自分のデータを使う方法）

右上の「⚙️（歯車アイコン）」から設定メニューを開き、自分のスプレッドシートを読み込ませることができます。

### スプレッドシートの準備
1. Googleスプレッドシートを新規作成します。
2. 1行目に以下のヘッダー（列名）を入力してください。
   - A列: `カテゴリ`
   - B列: `議題・話題（問いかけ例）`
   - C列: `選択回数` （任意、空欄でもOK）

| カテゴリ | 議題・話題（問いかけ例） |
| --- | --- |
| 軽い雑談 | 最近食べた美味しいものは？ |
| PC・IT | おすすめのスマホアプリは？ |
| マネー | 貯金してますか？ |

3. 2行目以降に、好きな話題を入力してください。
4. スプレッドシートの画面右上の「共有」ボタンを押し、「リンクを知っている全員」が「閲覧者」になるように設定し、リンクをコピーします。
   - または、URLバーのURL（`https://docs.google.com/.../edit...`）をコピーしてもOKです。

### アプリへの反映
1. アプリの設定画面（⚙️）を開きます。
2. 「スプレッドシートURL」の欄に、コピーしたURLを貼り付けます。
3. 必要に応じて「アプリタイトル」も変更できます（例：「◯◯カレッジ 雑談ガチャ」）。
4. 「保存する」を押すと、あなたのデータが読み込まれます。

## 🛠️ 開発者向け情報

### 技術スタック
- Framework: Next.js 16 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Data Fetching: Google Sheets (CSV Export)
- State Management: React Hooks + LocalStorage

### ローカルでの実行
```bash
git clone https://github.com/iidaatcnt/zatsudan-gacha-v2.git
cd zatsudan-gacha-v2
npm install
npm run dev
```

### デプロイ
Vercelへのデプロイを推奨します。
環境変数 `SHEET_CSV_URL` にデフォルトのスプレッドシートCSV URLを設定してください。
